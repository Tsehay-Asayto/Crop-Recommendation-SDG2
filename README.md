# Smart Crop Recommendation System (SDG 2: Zero Hunger)

## Project Overview This project addresses UN SDG 2: Zero Hunger by using Machine Learning to help farmers identify the most suitable crops for their land. By analyzing soil and weather parameters, the model ensures better yields and sustainable food production.

## Machine Learning Approach

Type: Supervised Learning (Classification)

Algorithm: Random Forest Classifier

Tools: Python, Scikit-learn, Pandas, Google Colab
